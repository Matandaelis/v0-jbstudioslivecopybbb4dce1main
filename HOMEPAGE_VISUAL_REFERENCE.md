## Animation Timings

\`\`\`
Float-up Animation
Duration: 2s
Easing: ease-out
Pattern: 
  0%:   opacity: 1, scale: 1, translateY: 0
  100%: opacity: 0, scale: 0.5, translateY: -100px

Fade-in Animation
Duration: 0.3s
Easing: ease-out
Pattern:
  0%:   opacity: 0, translateY: 10px
  100%: opacity: 1, translateY: 0

Slide-in Animation
Duration: 0.4s
Easing: ease-out
Pattern:
  0%:   opacity: 0, translateX: -20px
  100%: opacity: 1, translateX: 0

Pulse-ring Animation (Live Badge)
Duration: 2s
Easing: none (discrete steps)
Pattern:
  0%:   box-shadow: 0 0 0 0 rgba(primary, 0.7)
  70%:  box-shadow: 0 0 0 10px rgba(primary, 0)
  100%: box-shadow: 0 0 0 0 rgba(primary, 0)
\`\`\`

## Responsive Grid Changes

\`\`\`
Mobile (< 768px):
- Single column layouts
- 2x2 category grid
- Full-width cards
- Stacked sections

Tablet (768px - 1024px):
- 2 column grids for streams
- Maintain 2x2 categories
- Side layouts for preview
- Enhanced spacing

Desktop (> 1024px):
- 3 column grids for streams
- 4x1 category layout
- Side-by-side layouts
- Maximum content width
\`\`\`

## Spacing Scale

\`\`\`
Padding/Margin/Gap:
- xs: 0.25rem (4px)
- sm: 0.5rem (8px)
- md: 1rem (16px)
- lg: 1.5rem (24px)
- xl: 2rem (32px)
- 2xl: 2.5rem (40px)
- 3xl: 3rem (48px)

Common Patterns:
- Section padding: py-16 md:py-24
- Container padding: px-4 md:px-6
- Card padding: p-4
- Content gap: gap-6

Max Content Width: 1280px (7xl)
\`\`\`

## Visual Depth

\`\`\`
Shadows:
- Light: shadow-sm (subtle)
- Medium: shadow-lg (hover state)
- Heavy: shadow-2xl (prominent cards)

Border Treatments:
- Default: border-border (light gray)
- Hover: border-primary/20 or border-primary
- Active: border-primary (full color)

Opacity Layers:
- Full: 1 (visible)
- Focus: 0.9 (slight dim)
- Disabled: 0.5 (faded)
- Subtle: 0.1-0.2 (almost invisible)
\`\`\`

## Accessibility Visual Indicators

\`\`\`
Focus State (Keyboard Navigation):
┌──────────────┐
│ Button Text  │ ← Ring: 2px solid primary
└──────────────┘    Outline effect

Color Indicators:
🔴 Red: Errors, destructive actions
🟢 Green: Success, positive actions
🔵 Blue: Information, primary actions
🟠 Orange: Warnings, caution needed

High Contrast Text:
Dark text on light background (ratio 4.5:1+)
Light text on dark background (ratio 4.5:1+)
\`\`\`
